How I Code
My Approach
Architecture
SSA — Single Script Architecture

Instead of random scripts scattered across your game, SSA keeps everything in one organized place. Each feature lives in its own small module, and a single main script initializes everything. Cleaner project, faster fixes, easy to expand later.

Pattern
OOP — Object Oriented Programming

When something needs to repeat — like different stands or enemies — I use OOP. Write the logic once, create copies as needed. No messy duplication and the system scales perfectly for big games.

My Setup
Workflow
I code in Visual Studio Code for a superior development experience. I use Rojo to sync my scripts to Roblox in real-time, Selene to catch errors before they happen, and StyLua to keep every line of code perfectly formatted and readable.
