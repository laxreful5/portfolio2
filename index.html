About Me

I'm Ripper. Over the past 5 months and a half I've been scripting on Roblox every single day, pushing myself deep into Roblox Luau.


I take commissions to keep becoming better at scripting and to learn new concepts. My goal is to deliver solid results while constantly improving my skills.

What I Can Do

  • DataStore Handling (Saving stats, using ProfileStore)
  • Currency, Shops & Inventories
  • Simulator mechanics & Core loops
  • Player progression & Leveling
  • Bug fixing & Cleaning old code
  • Custom systems built for you

⚠️ Note: I do not take vehicle-related systems (advanced driving, car physics, etc.) at this time.

Tools I Use

I work with VSC, a famous tool editor that makes coding easier, and Rojo for communication between Roblox Studio and VSC. Wally for installing packages, Git to post work on GitHub for easier access when multiple scripters work on the same project, Selene as a linting extension, and StyLua to style your code to make it more readable.

My Workflow (SSA, OOP)

I use Single Script Architecture. Instead of having random scripts everywhere, I organize your game into clean modules.


This makes your game run smoother, makes bugs easier to find, and keeps everything organized as you add new features. I also use OOP (Object Oriented Programming).
