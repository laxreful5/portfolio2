Tools I Use

I work with VSC, a famous tool editor that makes coding easier, and Rojo for communication between Roblox Studio and VSC. My Workflow (SSA, OOP)

I use Single Script Architecture. Instead of having random scripts everywhere, I organize your game into clean modules.


This makes your game run smoother, makes bugs easier to find, and keeps everything organized as you add new features. I also use OOP (Object Oriented Programming).
